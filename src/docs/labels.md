# Labels

Optional tags for your entries — a way to filter the journal to one aspect of life at a time.

### When labels help

By default, everything flows into the same stream. Sometimes you want a narrower view: just the work thoughts during the work day, just the home stuff on a Saturday.

Tag work-related entries with a **Work** label, filter to it from the toolbar, and the rest drops away until you switch back.

### Creating and managing labels

Labels live in **Settings → Labels**. Create them, rename them, delete the ones you've outgrown.

When you delete a label, it's removed from any entry that had it — the entries themselves stay.

### Try this

- Keep them broad — Work, Family, Reading. Not Project Aurora Q3 retro.
- If a label stops being useful, just delete it.
