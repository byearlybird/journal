# Labels

Optional tags for your entries — a way to mark different aspects of life so you can filter the journal to one of them at a time. Most days, you won't need them.

### When labels help

The default mode is to let everything flow into the same stream — work thoughts, errands, half-formed ideas, all together. That's good. But sometimes you want a narrower view: just the work thoughts during the work day, just the home stuff on a Saturday morning. That's what labels are for.

Say you tag work-related entries with a **Work** label. Filter to it from the toolbar and the day's other noise drops away until you switch back.

### Creating and managing labels

Labels live in **Settings → Labels**. Create a few (or none), rename them when they no longer fit, delete the ones you've outgrown.

When you delete a label, it's removed from any entry that had it — the entries themselves stay.

### Try this

- Start with no labels. Add one only when you notice yourself wishing you could filter.
- Keep them broad — Work, Family, Reading. Not Project Aurora Q3 retro.
- Use them sparingly, not on every entry.

### Avoid this

- Treating labels as folders to organize everything into. The journal is meant to flow, not file.
- Spending time pruning and re-tagging. If a label stops being useful, just delete it.
